Uploading Data Into Table – Productivity & Utilisation Tracker
Overview
This Excel macro automates uploading data from an input section into a protected table. It is part of a department-wide Productivity & Utilisation Tracker, used daily by 60–80 colleagues across 9 versions for different teams via SharePoint.
Key Features & Benefits
Safe and concurrent usage: Prevents simultaneous macro execution, avoiding data corruption.
Safe exits: Ensures the macro completes successfully every time, maintaining table integrity.
Department-wide efficiency: Streamlines daily data input, reduces manual errors, and saves time.
Usage
Open the Excel workbook on SharePoint.
Enter data in the input section.
Click the assigned macro button to upload data into the protected table.
Technologies / Tools
Excel VBA (Macros)
SharePoint integration
Data protection and validation
Impact
Used daily by 60–80 team members, improving data accuracy and process reliability.
Eliminates the risk of table corruption due to concurrent macro execution.
